# 📊 Excel Reports Generator

Automação em Python para criação de relatórios de vendas a partir de arquivos CSV.

## 🚀 Funcionalidades

- Leitura automática de dados CSV
- Cálculo de faturamento
- Identificação do produto mais vendido
- Geração de relatório em Excel

## 🛠 Tecnologias

- Python 3
- Pandas
- CSV

## 📂 Arquivos

- generate_report.py → script principal
- vendas.csv → dados de exemplo
