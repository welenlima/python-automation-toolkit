# 🐍 Python Automation Toolkit

Coleção de scripts Python para automatização de tarefas repetitivas,
organização de arquivos e geração automática de relatórios.

## 🚀 Objetivo

Este projeto demonstra aplicações práticas de Python para aumentar
produtividade e reduzir tarefas manuais.

## 🛠️ Tecnologias utilizadas

- Python 3
- Pandas
- OpenPyXL
- Requests
- Automação de arquivos
